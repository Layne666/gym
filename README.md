# About

健身工作室后台管理系统

正在开发中，慢慢完善此项目！😉

注：此项目纯属个人项目，不用于任何商业用途。

# 说明

开发环境 Java8 + MySQL5.7  

部署环境 阿里云 CentOS 7.2 64位

如有问题请直接在 Issues 中提，或者您发现问题并有非常好的解决方案，欢迎 PR 👍

# 相关技术

## 后端
SpringBoot + SpringMVC + Mybatis 

## 前端

Amaze UI + JQuery + Vue.js 

## 渲染模板
Thymeleaf
